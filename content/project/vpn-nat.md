# How to configure access to computer behind NAT Network using OpenVPN.

## Requirements:
* Linux server with root access preferable CentOS 7
* Basic bash knowledge

### Let's start!


```bash
  ssh root@<server>
```

Make some updates

```bash
  yum -y update && yum -y upgrade
```
